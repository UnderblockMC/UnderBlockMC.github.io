Static website starter for GitHub Pages.

## Pages
- index.html
- features.html
- play.html
- rules.html
- store.html

## Main IP
- play.underblockmc.com

## Deploy to GitHub Pages
1. Create a new GitHub repository.
2. Upload all files from this folder.
3. In GitHub, go to Settings → Pages.
4. Set the source to the main branch root.
5. Add your custom domain, such as underblockmc.com.
6. Point your domain DNS to GitHub Pages.
